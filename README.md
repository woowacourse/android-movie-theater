# 3,4 단계 미션

## 기능 명세

- 네비게이션의 예매 내역 항목에 예매한 영화 리스트를 보여준다.
  - 예매한 영화 리스트 (티켓 리스트)
- 예매 내역을 터치하면 예매 정보를 보여준다.
  - 리스트 항목 클릭 시 press 효과를 주어야 한다.
- 앱을 재실행해도 기존 예매 내역이 유지되어야 한다.
  - Room Database를 활용해서 데이터를 관리한다.
- 설정에서 알림 기능을 On/Off 할 수 있다.
  - 사용자가 앱을 재실행해도 기존 설정 데이터가 남아있어야 한다.
- 영화 시작 시간 30분 전에 푸시 알림이 온다.
  - 푸시 알림을 클릭하면 예매 정보를 보여준다.
  - 알림 권한 Dialog, Notification은 기본 UI를 사용한다.
  - 아이콘은 커스텀해서 사용한다.

**ReservationHistoryFragment**

- 예약 티켓을 보여준다.
- 티켓을 선택할 수 있다.
- 티켓을 선택하면 티켓 상세 화면으로 이동한다.
  - 이동 시 티켓 정보를 전달한다.
- 티켓을 선택하면 press 효과를 볼 수 있다.

**SettingFragment**

- 앱의 기본 설정을 할 수 있다.
- 알림 기능을 On/Off 할 수 있다.
  - On 동작 시 푸시 알림을 허용한다.
  - Off 동작 시 푸시 알림을 거부한다.
  - 푸시 알림을 클릭하면 예매 상세 정보를 보여주어야 한다.
